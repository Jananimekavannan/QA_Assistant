# 🤖 QA Assistant

## 📌 Overview
The **QA Assistant** is an AI-powered chatbot that answers user questions in real time. It uses advanced language models to understand queries and generate clear, accurate responses.

This project is designed to help users quickly find answers, clarify doubts, and learn efficiently.

---

## 🚀 Features
- 💬 Interactive question-answering system  
- ⚡ Instant and accurate responses  
- 🧠 AI-powered understanding  
- 🌐 Simple and user-friendly interface  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- LangChain  
- Google Gemini API
